SETUP STEPS (Copy-Paste Into PowerShell)
════════════════════════════════════════════════════════════════════════

STEP 1: Copy files to your project
────────────────────────────────────
Copy these files from /mnt/user-data/outputs/ to C:\Projects\michael_command_center\
  • dashboard.html (updated to use Vercel backend)
  • vercel.json (config file)
  • api/claude.js (backend handler)

STEP 2: Navigate to your project
────────────────────────────────────
cd C:\Projects\michael_command_center


STEP 3: Install Vercel CLI
────────────────────────────────────
npm install -g vercel


STEP 4: Login to Vercel
────────────────────────────────────
vercel login

(Authenticate with your GitHub account)


STEP 5: Deploy to Vercel
────────────────────────────────────
vercel --prod

(Copy the URL shown, looks like: michael-command-center-xyz.vercel.app)


STEP 6: Add Claude API Key to Vercel
────────────────────────────────────
vercel env add CLAUDE_API_KEY

(Paste your sk-ant-* key when prompted)


STEP 7: Redeploy with the Key
────────────────────────────────────
vercel --prod


STEP 8: Update Dashboard Config
────────────────────────────────────
Edit dashboard.html and find this line:

  const VERCEL_URL = 'https://your-vercel-url.vercel.app';

Replace 'your-vercel-url' with your actual URL from step 5.


STEP 9: Push to GitHub
────────────────────────────────────
git add .
git commit -m "add vercel backend"
git push


STEP 10: Access Your Dashboard
────────────────────────────────────
Open: https://Mgregory2K.github.io/michael-command-center/dashboard.html

Now Grind Finder, PBJ, and all AI features work directly in the dashboard!

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════

If you get "API error" in dashboard:
  1. Check Vercel deployment status
  2. Verify CLAUDE_API_KEY environment variable is set
  3. Make sure you ran "vercel --prod" after adding the key
  4. Check browser console for detailed error (F12)

If Vercel deployment fails:
  1. Make sure you have Node.js installed
  2. Verify you're in the right directory (C:\Projects\michael_command_center)
  3. Check that api/claude.js and vercel.json exist
  4. Run "vercel --prod --debug" for detailed error

If dashboard won't load:
  1. Make sure VERCEL_URL in dashboard.html is correct
  2. Hard refresh browser (Ctrl+Shift+R)
  3. Check that GitHub Pages is still enabled
  4. Verify internet connection
